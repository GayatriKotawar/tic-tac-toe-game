Tic Tac Toe Game 

A simple Tic Tac Toe game built using HTML, CSS, and JavaScript.

Features
- Two player game (X vs O)
- Winner detection
- Draw detection
- Restart game option

Technologies Used
- HTML
- CSS
- JavaScript

How to Run
1. Clone the repository
2. Open `index.html` in any browser

Future Improvements
- Single player mode (AI)
- Scoreboard
- Better UI animations
